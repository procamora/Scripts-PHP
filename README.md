Recopilacion de pequeños scripts desarrollados en PHP

* **MonitorizaAntenas.php**
Saca los valores de todas las antenas que hay en el array y guarda los datos en una base de datos para su posterior analisis.

* **ConsultaInsertAsterisk.php**
Crea una consulta de mysql para insertar en la base de datos de asterisk una extension, en un futuro implementarlo para hacerlo con un for.
